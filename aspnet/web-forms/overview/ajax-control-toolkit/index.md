<a name="older-versions---ajax-control-toolkit"></a>以前のバージョンの Ajax コントロールのツールキット
====================
> ツールキットを使用して、Ajax コントロール、作業を開始する方法といくつかの最も一般的なコントロールを使用する方法。 DevExpress では、無料、オープン ソースの Ajax コントロール ツールキットを保持します。
> 
> 
> ![](index/_static/image1.png)DevExpress、受賞歴のある ASP.NET WebForms、MVC と HTML5 UI コントロールと、フレームワークの開発者は、この要求の高いオープン ソース ライブラリ信頼性の高い状態に戻すために、ASP.NET AJAX コントロール Toolkit の保守よりもかかりました。 [詳細については](https://go.devexpress.com/AjaxControlToolkit_ASP_Resources_Tutorials.aspx)します。
