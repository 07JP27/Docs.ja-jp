### <a name="web-server-controls"></a>Web サーバー コントロール

ほとんどの ASP.NET Web フォーム アプリケーションでは、ユーザーが、ボタン、テキスト ボックス、リスト、およびよびななど、ページと対話できるようにするコントロールを追加します。 これらの Web サーバー コントロールは、HTML ボタンおよび入力要素に似ています。 ただし、プロパティを設定してサーバー コードを使用できるように、サーバーで処理されます。 これらのコントロールでは、サーバー コードで処理できるイベントも発生します。

サーバー コントロールは、ページの実行時に ASP.NET がで認識される特殊な構文を使用します。 ASP.NET サーバー コントロールのタグの名前で始まり、`asp:`プレフィックス。 これにより、ASP.NET を認識し、これらのサーバー コントロールを処理できます。 プレフィックスは、コントロールは、.NET Framework の一部ではない場合は異なる可能性があります。 加え、`asp:`プレフィックス、ASP.NET サーバー コントロールも含める、`runat="server"`属性および`ID`サーバー コード内でコントロールの参照を行えます。

ページの実行時に ASP.NET サーバー コントロールを識別し、それらのコントロールに関連付けられているコードを実行します。 多くのコントロールの HTML またはその他のマークアップにレンダリング ページが表示されたらそれをブラウザーでします。
