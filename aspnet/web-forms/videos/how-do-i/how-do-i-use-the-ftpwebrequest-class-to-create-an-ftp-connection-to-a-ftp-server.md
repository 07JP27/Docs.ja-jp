<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[How Do i:]FtpWebRequest クラスを使用して、FTP サーバーに FTP 接続を作成するには
====================
によって[Chris Pels](https://twitter.com/chrispels)

このビデオ Chris Pels で FtpWebRequest クラスを使用して FTP 接続を作成し、ファイルを転送する方法を示しています。 まず、FTP サーバーに転送するファイルを選択する単純な .aspx フォームを作成します。 次に、重要なメソッドと FTP 接続を確立するために構成する必要がある FtpWebRequest クラスのプロパティを参照してください。 その後、StreamReader と StreamWriter を使用して、FtpWebRequest に書き込むには、ファイルを転送できるように、ローカル ファイルの内容を取得する方法を参照してください。 最後に、転送の結果を決定するのには、FtpWebResponse が取得されます。 この方法は、プログラムによって web アプリケーションのファイルの転送に使用できます。
