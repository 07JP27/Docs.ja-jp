# <a name="primitives-in-aspnet-core"></a>ASP.NET Core のプリミティブ

ASP.NET Core プリミティブは、フレームワーク拡張機能で共有される低レベルの構成ブロックです。 独自のコードでこれらの構成ブロックを使用できます。
