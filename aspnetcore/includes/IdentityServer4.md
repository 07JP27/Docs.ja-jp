ASP.NET Core Id を ASP.NET Core web アプリにユーザーインターフェイス (UI) のログイン機能を追加します。 Web Api と SPAs をセキュリティで保護するには、 [IdentityServer4](https://identityserver.io)を使用します。 IdentityServer4 は ASP.NET Core 3.0 用の OpenID Connect および OAuth 2.0 フレームワークです。 IdentityServer4 により、次のセキュリティ機能が有効になります。

* サービスとしての認証 (AaaS)
* 複数のアプリケーションの種類でのシングル サインオン/オフ (SSO)
* API のアクセス制御
* Federation Gateway

詳細については、「[ようこそ! IdentityServer4](http://docs.identityserver.io/en/latest/index.html)」を参照してください。
