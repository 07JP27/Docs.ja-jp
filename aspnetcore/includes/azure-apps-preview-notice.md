> [!IMPORTANT]
> **Azure App Service での ASP.NET Core 3.0**
>
> ASP.NET Core 3.0 は、まだ Azure App Service に展開されていません。 間もなく Azure App Service 上で ASP.NET Core 3 を提供できるようになる予定です。 ASP.NET Core 3.0 を使用するアプリをホストするには:

* Azure App Service の展開のプレビュー リリースと同様に ASP.NET Core 3.0 を扱います。
* 「[Azure App Service に ASP.NET Core プレビュー リリースを展開する](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service)」を参照してください。
