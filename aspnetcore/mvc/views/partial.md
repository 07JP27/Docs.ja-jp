## <a name="partial-view-discovery"></a><span data-ttu-id="b7d53-161">部分ビューの検出</span><span class="sxs-lookup"><span data-stu-id="b7d53-161">Partial view discovery</span></span>

<span data-ttu-id="b7d53-162">部分ビューを参照するときに、いくつかの方法でその場所を参照することができます。</span><span class="sxs-lookup"><span data-stu-id="b7d53-162">When referencing a partial view, you can refer to its location in several ways.</span></span> <span data-ttu-id="b7d53-163">例:</span><span class="sxs-lookup"><span data-stu-id="b7d53-163">For example:</span></span>

::: moniker range=">= aspnetcore-2.1"

```cshtml
// Uses a view in current folder with this name.
// If none is found, searches the Shared folder.
<partial name="_ViewName" />

// A view with this name must be in the same folder
<partial name="_ViewName.cshtml" />

// Locate the view based on the app root.
// Paths that start with "/" or "~/" refer to the app root.
<partial name="~/Views/Folder/_ViewName.cshtml" />
<partial name="/Views/Folder/_ViewName.cshtml" />

// Locate the view using a relative path
<partial name="../Account/_LoginPartial.cshtml" />
```

<span data-ttu-id="b7d53-164">前の例では、ASP.NET Core 2.1 以降を必要とする部分タグ ヘルパーを使用しています。</span><span class="sxs-lookup"><span data-stu-id="b7d53-164">The preceding example uses the Partial Tag Helper, which requires ASP.NET Core 2.1 or later.</span></span> <span data-ttu-id="b7d53-165">次の例では、非同期の HTML ヘルパーを使用して、同じタスクを実行しています。</span><span class="sxs-lookup"><span data-stu-id="b7d53-165">The following example uses asynchronous HTML Helpers to accomplish the same task.</span></span>

::: moniker-end

```cshtml
// Uses a view in current folder with this name.
// If none is found, searches the Shared folder.
@await Html.PartialAsync("_ViewName")

// A view with this name must be in the same folder
@await Html.PartialAsync("_ViewName.cshtml")

// Locate the view based on the app root.
// Paths that start with "/" or "~/" refer to the app root.
@await Html.PartialAsync("~/Views/Folder/_ViewName.cshtml")
@await Html.PartialAsync("/Views/Folder/_ViewName.cshtml")

// Locate the view using a relative path
```
